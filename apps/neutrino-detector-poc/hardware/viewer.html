KiCad Viewer

Live PCB renders · v0.1 skeletons

Serving locally. Renders .kicad_pcb files from each board directory directly in the browser via a small S-expression parser. To launch: 
cd apps/neutrino-detector-poc/hardware
python3 -m http.server 8765
Then open http://localhost:8765/viewer.html. When Adam saves a real schematic over the skeleton, this view updates on refresh. Drag-and-drop a .kicad_pcb onto any board for a transient preview.
