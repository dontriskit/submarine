Why MCP

CLI
Insufficient.

Fork overhead per command. No streaming. Breaks under continuous sensor feedback.

REST
Partial.

Request and response. Polling wastes energy. State doesn't persist between calls.

MCP
Persistent.

Bidirectional. Streaming sensor data. State lives in the server. Purpose-built for agents.
