by 2036 i'll build a submarine.

PROTOTYPE IN DEVELOPMENT — v0.1

Mission

A vessel that can explore, transport, extract, search, guide, help, rescue, and clean. Controlled by AI. Open source. Built to bridge gaps between continents and clean the oceans before it is too late.

🌊

Explore

Map ocean floors, collect water samples, discover what's down there.

🗑️

Clean

Detect and extract floating debris. Autonomous trash collection at scale.

🚁

Launch

Deploy drones from below. Extend reach above and below the surface.

🧠

Think

AI agent swarm. Each arm is an autonomous agent. The head orchestrates.

Architecture

Each arm is an independent AI agent instance. The head orchestrates — decomposing tasks, resolving conflicts, aggregating results. Arms share memory files on disk but maintain isolated context windows. MCP for persistent bidirectional communication.
